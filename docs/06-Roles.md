# System Roles

## Patient
- Request medical consultations
- Access psychological support
- Request medications and equipment
- Manage personal profile and feedback

## Doctor / Counselor
- Provide consultations and counseling
- Manage availability
- Verify patient medical cases
- Create treatments and campaigns

## Donor
- Browse sponsorship campaigns
- Donate to treatments
- Track donation usage

## NGO Staff
- Manage organizations and inventory
- Coordinate aid operations
- Report fund disbursement

## Volunteer
- Deliver medications and equipment
- Update delivery status

## Admin
- Oversee platform operations
- Audit donations and campaigns
- Manage system-wide settings
