# Technologies & Tools

## Backend Technologies
- Node.js: JavaScript runtime environment
- Express.js: RESTful API framework
- MySQL: Relational database management system

## Security & Authentication
- JWT (jsonwebtoken): Authentication and role-based access control
- bcryptjs: Password hashing
- Helmet: HTTP security headers
- express-rate-limit: Protection against brute-force attacks
- CORS: Cross-origin request handling

## Validation & Utilities
- fastest-validator: Request validation
- body-parser: Request body parsing
- Multer: File uploads
- node-cron: Scheduled background tasks
- Nodemailer: Email notifications
- Axios: External API communication

## Documentation & Testing
- Postman: API testing and documentation
- YAML: OpenAPI specifications
- GitHub: Version control and collaboration

## External Services
- OpenAI API: Medical text translation
- Daily API: Webinar and online meeting generation
- PayPal: Secure donation processing for medical sponsorship campaigns using a two-step payment flow to ensure payment verification and transparency.
