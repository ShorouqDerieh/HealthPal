## Overview
Feature 5 provides secure psychological support services for victims of war,
trauma, children, and individuals with PTSD.

The feature emphasizes privacy, anonymity, and secure handling of sensitive
psychological data.

---

## Core Components

### 1. Trauma Counseling Portal
- Creation and management of counseling sessions
- Patients and doctors can view their sessions
- Sessions can be canceled when needed
- Counseling notes are encrypted using AES-256-CBC

### 2. Support Groups
- Creation and management of psychological support groups
- Users can request to join groups
- Group admins manage members
- Members can send messages within groups

### 3. Anonymous Therapy Chat
- Anonymous one-on-one chat sessions
- Automatic pseudonym generation
- Secure message exchange
- Reduces stigma associated with therapy

---

## Security & Privacy
- AES-256-CBC encryption for counseling notes
- Anonymous identity handling
- Access restricted to authorized participants only

---

## Roles Involved
- Patient
- Counselor / Doctor
- Admin
